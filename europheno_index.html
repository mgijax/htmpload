Europhenome/MP Annotation Load: Inputs, Logs, etc.

    Inputs/Outputs

    • Europhenome BioMart file (contains both MP-annotated & non-MP-annotated rows)
    • Europhenome BioMart re-formatted to HTMP file
    • HTMP file (contains MP-annotated rows only)
    • HTMP duplicate file (male/female duplicates)
    • HTMP skipped file (rows where MP id is blank)
    • HTMP error file (errors that can are also reported in curator log)
    • MGI-format: Genotype (contains unique genotypes; field 2 contains existing genotype IDs)
    • MGI-format: Genotype (new) (contains unique genotypes; field 2 contains both new and existing genotype IDs)
    • MGI-format: Annotation

    Diagnostic and Load BCP Files

    • europhenoload.diag.log
    • europhenoload.cur.log
    • mgi_genotypeload.txt.diagnostics
    • mgi_genotypeload.txt.error
    • mgi_annotload.txt.*.diagnostics (file name contain date stamp)
    • mgi_annotload.txt.*.error (file name contain date stamp)
