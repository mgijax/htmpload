Sanger/MP Annotation Load

    Inputs/Outputs

    • Sanger BioMart file (contains both MP-annotated & non-MP-annotated rows)
    • HTMP file (contains MP-annotated rows only)
    • HTMP duplicate file (male/female duplicates)
    • HTMP skipped file (rows where MP id is blank)
    • HTMP error file (errors that are also reported in curator log)
    • MGI-format: Genotype (contains unique genotypes; field 2 contains existing genotype IDs)
    • MGI-format: Genotype (new) (contains unique genotypes; field 2 contains both new and existing genotype IDs)
    • MGI-format: Annotation
