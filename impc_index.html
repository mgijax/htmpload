    Mammalian Phenotype:Input Files

    • Intermediate file of IMPC data with versioning duplicates removed (1c)

    • HTMP load format file created from source IMPC/iMits2 data (1e)
    • IMPC Strain file created from source IMPC/iMits2 data (1f)
    • HTMP error file - Input lines for errors that are reported to the curation log (1g)
    • Lines missing required data (1h)
    • HTMP Unique Genotype file - Contains unique genotypes; field 2 contains new genotype IDs (3a)
    • HTMP duplicate file - male/female records collapsed to 'both'(3b)
    • MGI Genotype load format file - Contains unique genotypes; field 2 contains existing genotype IDs (3c)
    • MGI Genotype load format file (new) - Contains unique genotypes; field 2 contains both new and existing genotype IDs (3d)
    • MGI Annotation load format file (4a)
