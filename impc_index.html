    • IKMC iMits2 BioMart file - Used to get MCL ID; joined to IMPC JSON file using colony ID

    Intermediate Input Files

    • Intermediate file of IMPC JSON file data with versioning duplicates removed - Tab-delimited (2319)
    • The versioning duplicates from IMPC JSON file - Tab-delimited (413)
    • HTMP duplicate file - male/female duplicates (729)
    • HTMP skipped file - Rows where MP id is blank (0)

    • HTMP load format file
    • MGI Strain load format file - contains all strains from HTMP format file(308, 36 already exist)

    • MGI Genotype load format file (new) - Contains unique genotypes; field 2 contains both new and existing genotype IDs (335)
    • MGI Annotation load format file (1422)
